# Personal-stock-gui 使用說明書

本文件將為您詳細介紹 Personal-stock-gui 系統中各個獨立頁面的核心功能與使用情境。透過本指南，您與您的團隊可以更快速地掌握這套個人化股票量化分析與預測系統的強大功能。

## 頁面功能總覽與詳細說明

### 1. 儀表板 (Dashboard)
- **功能概述**：系統狀態與市場全貌的總覽中心。
- **使用情境**：開啟系統後的第一個畫面，讓您快速檢視目前系統追蹤的熱門股票概況、最新的預測數據摘要以及整體市場的動態概覽，讓您能在最短時間內掌握最新的盤勢狀況。

### 2. 觀察名單 (Watchlist)
- **功能概述**：管理並追蹤您個人關注的股票清單。
- **使用情境**：允許您新增、移除與追蹤特定的股票代碼。系統會在此頁面集中顯示您清單內股票的即時報價變動與關鍵技術指標，方便您每日進行例行性檢查。

### 3. 股票分析 (Stock Analysis)
- **功能概述**：單檔股票的深度技術與籌碼分析介面。
- **使用情境**：當您需要深入研究某一檔特定的股票時，輸入代碼後，系統會載入該股票的歷史 K 線圖，並允許您疊加各項常見的技術指標 (如 RSI、MACD、布林通道等)。這是您進行基本量化分析與人工判讀的主要工具。

### 4. 技術分析報告 (TA Report)
- **功能概述**：自動生成並檢閱 Markdown 格式的完整技術分析報告。
- **使用情境**：系統會根據最新的市場數據自動計算並產生綜合的技術分析報表，內容包含趨勢判斷與指標解讀。這個頁面非常適合用來直接匯出報告、作為小組簡報的素材或是個人的投資筆記。

### 5. KRONOS 模型預測分析 (KRONOS XAI)
- **功能概述**：展示 KRONOS 時間序列模型的專屬預測結果。
- **使用情境**：提供具備可解釋性人工智慧 (XAI) 技術的圖表。該頁面不僅會顯示未來股價的漲跌機率 (Up Probability) 與信心分數，還會透過視覺化指出模型做出該預測的依據與關鍵特徵。

### 6. TimesFM 模型預測分析 (TimesFM XAI)
- **功能概述**：展示 Google TimesFM 基礎模型的平行預測結果。
- **使用情境**：利用強大的時間序列基礎模型進行走勢推估，提供另一種維度的未來價格路徑預測與信賴區間。通常用來作為 KRONOS 預測結果的交叉驗證參考。

### 7. 模型預測比較 (Prediction Compare)
- **功能概述**：將多種預測模型的結果並列比較與分析。
- **使用情境**：此頁面會將 KRONOS 與 TimesFM 等模型的預測曲線重疊顯示，讓您能直觀地比較不同模型對未來看法的差異，從中尋找預測共識最強、勝率最高的交易訊號。

### 8. 模型校正 (Calibration)
- **功能概述**：針對預測引擎進行參數微調與準確度校正。
- **使用情境**：讓進階使用者能根據最新的市場環境變化或特定市場特性，重新校準模型的權重或判定閾值，確保預測引擎始終保持在最佳狀態。

### 9. 交易決策 (Trading Decision)
- **功能概述**：結合人工判斷與各項模型數據的綜合決策中心。
- **使用情境**：在此頁面中，您可以檢視各項分析與預測的總結，並設定您的交易部位規模、停損停利點與風險控制參數。系統會基於您的設定提供具體且量化的下單建議。

### 10. 多智能體模擬分析 (MiroFish Sim)
- **功能概述**：驅動 TradingAgents 框架進行虛擬分析師的深度辯論。
- **使用情境**：啟動後，系統會指派「市場」、「情緒」、「新聞」與「基本面」等四位基於大型語言模型 (LLM) 的虛擬分析師，針對當前股票進行多回合的辯論與探討。最後由虛擬投資經理產出綜合報告，為您揭示潛在的市場盲點、風險與機會。

### 11. 歷史回測 (Backtest)
- **功能概述**：驗證交易策略在過去真實市場中的表現。
- **使用情境**：設定特定的歷史時間區段與初始資金，系統將模擬使用當前的 AI 分析與決策邏輯進行虛擬交易，並產出包含總報酬率、最大資金回撤 (Max Drawdown) 與勝率的詳細績效圖表，協助優化策略。

### 12. 系統設定 (Settings)
- **功能概述**：管理系統的全局設定與 API 金鑰環境。
- **使用情境**：您可以在此切換系統的介面外觀、設定預設的資料來源與頻率，並且選擇您偏好的大型語言模型 (LLM) 供應商 (如 DeepSeek、OpenAI 等)，確保系統與外部服務的連線暢通。
