# KRONOS XAI Agent

## 專案簡介
KRONOS XAI Agent 是一個強大且高度視覺化的多智能體股票分析與交易模擬系統。本系統結合了大型語言模型 (LLM) 以及先進的時間序列預測模型 (TimesFM 與 KRONOS)，透過不同的專業虛擬分析師 (包含市場分析、情緒分析、新聞分析與基本面分析) 進行辯論與風險評估，並將結果呈現在由 PyQt6 打造的圖形化介面中。

## 核心功能
*   **多智能體辯論與分析**：系統內建多位專屬分析師，能夠根據當前的市場資料、新聞與技術指標進行深度的多方辯論，最終由投資組合經理做出統合決策。
*   **歷史回測與風險管理**：具備回測引擎與風險控管機制，能夠檢驗過去的預測準確度並控制未來決策的潛在風險。
*   **圖形化操作介面**：使用 PyQt6 建立直覺且資訊豐富的儀表板，讓使用者能輕鬆觀察模型預測曲線、技術指標變化以及各智能體的決策貢獻度。
*   **報告自動生成與匯出**：能將複雜的分析與辯論過程匯出成 Markdown 格式的完整報告，方便團隊檢閱與進行後續的簡報。

## 系統架構與目錄說明
專案主要目錄與檔案功能如下：

*   `ui/`：圖形化介面相關程式碼，包含儀表板與各種設定頁面。
*   `data/`：資料收集模組與本機 SQLite 資料庫 (`stock_advisor.db`)，負責儲存股票歷史資料與分析紀錄。
*   `decision/`：交易決策邏輯、風險管理與基於規則的輔助判斷模組。
*   `backtest/`：歷史回測引擎，用以驗證策略有效性。
*   `trading_agents/`：多智能體核心整合，負責連接外部 LLM 執行智能體辯論。
*   `reports/`：儲存由系統自動生成的股票技術分析與投資報告。
*   `utils/`：系統公用程式，包含日誌記錄 (logger)、通知與匯出工具。
*   `main_gui.py`：系統的主要啟動程式碼。

## 環境建置與安裝指南

### 1. 下載專案與建立虛擬環境
首先將專案複製到本機端，並建議建立一個 Python 虛擬環境以確保套件不會與系統其他專案衝突。

### 2. 安裝依賴套件
專案已提供完整的 `requirements.txt`。請在終端機中執行以下指令以安裝所有必要的套件：
```bash
pip install -r requirements.txt
```

### 3. 設定環境變數
系統運行需要連接外部的 LLM 服務 (例如 DeepSeek 等 API 金鑰)。請在專案根目錄下建立一個名為 `.env` 的檔案，並填入你所使用的服務金鑰：
```env
DEEPSEEK_API_KEY=你的金鑰填寫於此
OPENAI_API_KEY=你的金鑰填寫於此
# 根據你實際選擇的服務提供商填寫對應的金鑰
```
*(請放心，`.env` 已經加入 `.gitignore`，不會被上傳到公開的程式碼儲存庫中。)*

## 使用教學

確保所有依賴套件已安裝且環境變數設定完畢後，在終端機輸入以下指令啟動系統圖形化介面：

```bash
python main_gui.py
```

啟動後，你可以在左側的側邊欄切換不同的功能頁面，包含觀看大盤走勢、設定 API 來源，以及啟動多智能體模擬分析。所有產生的報表都會自動存入 `reports/` 資料夾中供後續使用。
